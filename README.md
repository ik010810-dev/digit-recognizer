# Digit Recognizer

Приложение для распознавания рукописных цифр с интерактивной визуализацией работы сверточной нейросети. Рисуй цифру — смотри как модель «думает».

## Возможности

- Рисование цифр мышкой на холсте
- Распознавание с показом уверенности
- Визуализация обработанного изображения (28×28) — что реально видит модель
- Карты активации последнего сверточного слоя (8 фильтров) — на какие признаки реагирует
- Послойная визуализация — как меняется «понимание» от слоя к слою
- Кнопка очистки холста
- Сохранение модели с автоматическим EarlyStopping

## Использование

```bash
python app.py
```

1. Нарисуй цифру в левой части окна
2. Нажми «Распознать»
3. Смотри результат и визуализацию справа:
   - **Верхний ряд** — сжатое изображение 28×28, которое подается в модель
   - **Средний ряд** — 8 карт активации последнего слоя. Красный = нейрон «загорелся», синий = не отреагировал
   - **Нижний ряд** — эволюция признаков по слоям. От контуров (слева) к абстрактным признакам (справа)
4. Нажми «Очистить» чтобы нарисовать новую цифру

## Архитектура модели

```
Input (28×28×1)
  → RandomRotation + RandomTranslation + RandomZoom (аугментация)
  → Conv2D(32) → BatchNorm → ReLU
  → Conv2D(32) → BatchNorm → ReLU → MaxPooling
  → Conv2D(64) → BatchNorm → ReLU
  → Conv2D(64) → BatchNorm → ReLU
  → GlobalAveragePooling
  → Dense(128) → Dropout(0.5)
  → Dense(10) → Softmax
```

## Обучение модели

Параметры обучения:
- **Оптимизатор:** Adam (learning_rate=0.001)
- **Функция потерь:** Sparse Categorical Crossentropy
- **Batch size:** 64
- **EarlyStopping:** patience=5, min_delta=0.001
- **Данные:** MNIST (60 000 картинок, 80%/20% train/val split со стратификацией)

## Как читать визуализацию

### Карты активации (8 фильтров)
Каждый фильтр ищет свой признак: горизонтальные линии, вертикальные, диагонали, петли, углы, круги.

- **Красный** — признак найден, нейрон сильно активирован
- **Синий** — признак отсутствует

### Послойная визуализация
Слева направо — от простого к сложному:
- **C1-R1:** Контуры и края цифры
- **C2-R2:** Части цифры (палки, изгибы)
- **C3-R3:** Абстрактные признаки (уменьшенные в 2 раза)
- **C4-R4:** Высокоуровневые признаки перед классификацией
